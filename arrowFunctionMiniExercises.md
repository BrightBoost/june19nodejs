# Arrow Function Mini Exercises

---

## Exercise 1 – Addition: Regular vs Arrow

Write an `add` function that takes `a` and `b` and returns their sum.
First write it as a **regular function**, then rewrite it as an **arrow function**.


---

## Exercise 2 – Higher-Order Function: `operate`

Write a function called `operate` that:

- accepts two numbers (`a`, `b`) and a **function** (`fn`)
- calls `fn` with `a` and `b` and returns the result


---

## Exercise 3 – Call `operate` with Two Different Math Arrow Functions

Using the `operate` function from Exercise 2, call it twice:

1. Once with an arrow function that **multiplies** `a` and `b`
2. Once with an arrow function that **subtracts** `b` from `a`
